Double Bubble: A Coding Project

In the vein of Fizz Buzz, if you DoubleBubble, you're in trouble!
Double Bubble is a JavaScript coding exercise to practice fundamentals.

Instructions

Enter a double value and a bubble value, and watch the table populate an array from 0-100, where the numbers that are multiples of the double and bubble value displays the words "Double" and "Bubble" respectively, instead. Numbers that are multiples of both, display "Double Bubble!" instead.

Enter the desired values and try it out!
